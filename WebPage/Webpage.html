👤 Section A: Contacts

🕵️ When to Create a Contact?

Create for the following:

  • TL-provided Cold Calling Data (only if prospect is a good fit)
  • Referral Customers
  • WhatsApp Leads (given by TL or Manager)

🔍 All Contacts Tab

View/search all contacts. Check for duplicates before adding new ones. Newly added contacts appear at the top.

📝 How to Add a Contact

Fields to Fill:

  • Customer Type → Customer
  • Contact Type → Visitor
  • Branch → Global Team
  • Assignee → Self
  • Rating:
    • ❄️ Cold = 0–40
    • 🌤️ Warm = 41–70
    • 🔥 Hot = 71+
  • Remark → Include requirement, budget, locality, updates
  • Source → Lead origin

🔄 Convert Contact to Lead

Steps:

  • Click the ➡️ icon next to the contact.
  • Fill in:
    • Comment → Customer requirement
    • Interested In → Project name
    • Schedule Date/Time → (Pre-filled by CRM)
    • Rating → Update if necessary

🚀 Section B: Leads

📂 Types of Leads

  • All Leads → Leads added or assigned to you
  • Today's Follow-ups → Leads with follow-ups due today
  • Backlogs → Missed follow-ups from previous days

🛠️ Changing Lead Status

New → No action taken yet

In Progress → Work like follow-ups, SV, VC, F2F is ongoing

Disqualified → Not interested; reasons include:

  • Invalid Number
  • Not Looking
  • Budget/Location Mismatch
  • Fake Lead or Competitor

Qualified → Booking or Token done

How to Change Status:

Tap the ☰ “Action” → Select Change Status → Choose appropriate status and add a Closing Remark

🔁 Scheduling Follow-ups

Click 🕗 icon and fill:

  • Current Update → Pick appropriate
  • Purpose → Pick appropriate
  • Note → Use simple language; include requirement, budget, location
  • Schedule Date/Time → For next follow-up
  • Score → Optional update

📞 Section C: Prospects

🤔 What Are Prospects?

Prospects are customers from cold calling data.
They are potential buyers, but we don’t yet know if their needs match any available project configurations.
Your job is to call them, check their requirements, and verify if they’re prospective customers.

🔄 Convert Prospect to Lead

If a prospect is interested and matches a project:

  1. ➡️ Click "Convert to Lead" (arrow icon)
  2. 📝 Fill in the form:
    • Comment → Mention requirement, budget, locality, and project name pitched
    • Interested In → Add project name
    • Schedule Date & Time → Leave default
    • Source → Cold Calling
    • Branch → Global Team
    • Assignee → Self
    • ✅ Select Source → Click Convert to Lead

Then go to All Leads, update the lead's status, add remark, and schedule a follow-up.

❌ Change Status (if not interested)

If prospect is not interested, you must update their status:

  • ☰ Tap the three-bar menu → Click Change Status
  • ✔️ Select appropriate status (e.g. Not Interested, Wrong Number, etc.)
  • 🗒️ Add a remark for disqualification
  • ✅ The prospect will be removed from your account after this.

💼 Section D: Opportunities

✍️ When to Create an Opportunity?

After:

  • ✅ VC Completed
  • ✅ F2F Completed
  • ✅ Token Done
  • ✅ Booking Done

📜 All Opportunities Tab

View all opportunities you’ve created.

➕ Creating a New Opportunity

Fields:

  • Customer → Contact or Lead
  • For → Buy
  • Looking For → Residential Apartment
  • Purpose/Stage → Choose from:
    • F2F → Office Visit
    • VC → General Follow-up
    • Token → Finalization
    • Booking → Completed
  • Remark → Include:
    • Reason (e.g. VC Done)
    • Project Name
    • Customer Requirement
    • Latest Comment
    • Optional: Flat Number + Booking/Token ID

Examples:

  • 🔹 VC Done at Godrej Evergreen Square, 2BHK interest, awaiting family discussion.
  • 🔹 Booking done at Godrej Evergreen Square, 2BHK, Flat 1307 Tower B, Booking ID: GPLES1201
  • Source → Lead source
  • Branch → Global Team
  • Assignee → Self
